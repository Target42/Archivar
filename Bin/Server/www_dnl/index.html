Den Launcher (Speichern unter)auf die Platte kopieren, Auspacken und starten. Dann wird das eigentliche Download-Tool gestartet.

Name Daten
Host <#xxhost>
ip <#xxhostip>
TCP/IP-Port <#xxtcpport>
HTTP-Port <#xxhttpport>
HTTPS-Port <#xxhttpsport>

Nach einem normalen Setup haben die Benutzer kein Passwort. Außer der Admin, wenn dem eines während der Installation erteilt wurde.

Verbindungen:
  • ds://<#xxhost>:<#xxtcpport>
  • http://<#xxhost>:<#xxhttpport>
  • https://<#xxhost>/<#xxhttpsport>

Das Archiv mit dem Server Server.zip runterladen und auspacken. Danach das Setup.exe ausführen um den Server einzurichten.

Die Anleitung als PDF Archivar.pdf

Achtung Einige Virenscanner (u.a. der Bitdefender) blockieren den Download des Programmes, bzw. löschen nach dem Download die Programmdateien. Daher ist zu empfehlen solche Tools zu Deaktivieren. Für einen ersten Installationsversuch ist die "Windows Sandbox" die ideale Wahl.
Hier ein Setup für den Test in der Windows-Sandbox: SandboxInstaller
Windows 11 verweigert oft den Start von nicht signierten Programmen aus dem Internet. In den Eigenschaften des Programmes kann man das Programm als Vertrauenswürdig erklären.
